🧬 Understanding Protein Synthesis

Learn about the fascinating process that transforms genetic information into functional proteins.

🔄

Transcription

This is like making a copy of DNA instructions. The cell copies DNA into mRNA (messenger RNA).

What happens:

  • DNA opens up like a zipper
  • A special enzyme reads the DNA code
  • It makes an mRNA copy of the instructions
  • The mRNA copy can leave the nucleus
  • DNA letters change: A→U, T→A, C→G, G→C
🏭

Translation

This is like using a recipe. The cell reads mRNA instructions and builds proteins from amino acids.

What happens:

  • Ribosomes (protein factories) read the mRNA
  • They read 3 letters at a time (called codons)
  • Each codon tells which amino acid to add
  • AUG means "start making protein"
  • Special stop signals mean "protein is finished"
🧩

Codon Usage

Codons are like words in the genetic language. Each "word" is 3 letters long and tells the cell which amino acid to use.

Important facts:

  • There are 64 possible 3-letter combinations
  • 61 of them mean "add this amino acid"
  • 3 of them mean "stop making the protein"
  • Most amino acids have more than one codon
  • AUG is special - it means both "start" and "add methionine"
  • Almost all living things use the same genetic code
🔗

Protein Folding

After amino acids are linked together, they fold into specific 3D shapes. The shape determines what the protein can do.

How proteins get their shape:

  • Amino acids are like beads on a string
  • The string folds and twists in specific ways
  • The final shape is like a complex 3D puzzle
  • Different shapes do different jobs in the cell
  • Special helper proteins help with folding
  • Wrong folding can make proteins not work
🔬

Cell Type Differences

There are two main types of cells, and they make proteins differently!

Animal/Plant Cells (Eukaryotes):

  • Have a nucleus (control center)
  • Make mRNA inside the nucleus
  • Make proteins outside the nucleus
  • Only use AUG to start making proteins
  • Process mRNA before using it

Bacterial Cells (Prokaryotes):

  • No nucleus - everything mixed together
  • Make mRNA and proteins in the same place
  • Can use AUG, GUG, or UUG to start proteins
  • Use mRNA right away without processing
  • Can make proteins while still copying DNA
⚡

Mutations

Sometimes DNA gets changed by mistake. These changes can affect how proteins are made.

Types of DNA Changes:

  • Substitution: One DNA letter becomes a different letter
  • Insertion: Extra DNA letters get added
  • Deletion: Some DNA letters get removed

What can happen to proteins:

  • Silent: Protein stays exactly the same
  • Missense: One amino acid changes
  • Nonsense: Protein stops being made too early
  • Frameshift: Reading frame shifts, everything changes
🧵

Understanding DNA Strands

DNA has two strands that work together, but they have different jobs during protein making.

The Two Strands:

  • Template Strand: This is the strand the cell actually reads to make mRNA
  • Coding Strand: This strand has the same sequence as mRNA (except T instead of U)
  • The two strands are like opposite sides of a zipper
  • They run in opposite directions (antiparallel)
  • Where one has A, the other has T; where one has C, the other has G

Why this matters:

  • Only the template strand gets read during transcription
  • The coding strand "shows" what the mRNA will look like
  • Scientists often write DNA sequences using the coding strand
  • This calculator lets you input either strand type
